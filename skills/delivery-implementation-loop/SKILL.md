## Loop

1. read the current plan and handoff dossier
2. implement the smallest shippable slice
3. run verification
4. publish what changed and what remains
5. hand off to review

## Rules

- do not expand scope beyond the approved MVP slice
- do not self-approve release readiness
- every handoff must point to tests, artifacts, or evidence
