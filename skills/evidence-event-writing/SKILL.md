# Evidence Event Writing

## Purpose

Record what happened, what was observed, and how much decision weight it has.

## Required Fields

- source
- source ref
- metric
- observed value
- threshold
- result
- confidence
- signal strength
- payment proximity
- cost
- limitations

## Rule

Blocked execution states are evidence about operational readiness, not market
validation.
