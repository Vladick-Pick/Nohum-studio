# Assumption Mapping

## Purpose

Identify what must be true for a product bet to deserve the next step.

## Assumption Types

- `buyer`
- `pain`
- `payment`
- `channel`
- `activation`
- `retention`
- `build`
- `legal`
- `economics`

## Output

Use `docs/templates/product-bets/assumption-map.md`.

Select one riskiest assumption and explain what evidence would support or
contradict it.
