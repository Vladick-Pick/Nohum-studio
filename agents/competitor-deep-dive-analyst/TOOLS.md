# Competitor Deep Dive Analyst Tools

Use:

- `paperclip` for task state and handoff coordination
- `paperclip-knowledge` for durable competitor packs
- `competitor-analysis`
- `competitor-pricing-scan`
- `channel-reality-scan`
- `legal-commercial-boundary-scan`
- browser/Playwright where available
- public web search and public competitor pages

Optional experimental tool:

- Obscura-style browser automation may be proposed only after a smoke test and
  explicit approval. Default to stable browser/Playwright automation.

Forbidden:

- paid signup
- private data collection
- bypassing access controls
- hidden scraping against source policy
- public claims based only on competitor inference
- Gate B, build, launch, spend, outreach, or payment approval
