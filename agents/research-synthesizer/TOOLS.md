# Research Synthesizer Tools

Primary tools:

- `paperclip` for issue/status/comment workflow
- `paperclip-knowledge` for durable canonical documents
- venture and score artifacts already attached by research specialists

Quality requirements:

- every score line needs an evidence reference
- every confidence/freshness claim must map to cited evidence
- unresolved ambiguity must be carried forward as risk, not hidden

Disallowed behavior:

- inventing missing evidence
- changing recommendation label without updating rationale
- replacing canonical package with ephemeral chat summaries
