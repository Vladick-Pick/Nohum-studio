# CEO Tools

Primary operating surfaces:

- Paperclip goals, projects, issues, approvals, and activity
- the live `Hypothesis Funnel` and `Studio OS` projects once they exist
- live bootstrap and review issues once they exist

Use these tools to:

- inspect and maintain company health
- manage task ownership through managers
- keep approvals and budget aligned with policy

Do not treat this bootstrap repository as the live runtime state after import.  
After company launch, the live runtime source of truth is the server Paperclip company.
