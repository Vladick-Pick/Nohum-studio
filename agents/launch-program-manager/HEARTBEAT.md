# Launch Program Manager Heartbeat

Run this checklist on every wake.

1. Re-open the venture brief, research dossier, and latest Gate B gaps.
2. Verify ICP, offer, pricing, UX, and scope preconditions before producing anything new.
3. Update the canonical launch packet, not just an isolated note.
4. Escalate unresolved definition gaps to `Launch Lead` early.
5. Stop only when the next owner and artifact are explicit.
