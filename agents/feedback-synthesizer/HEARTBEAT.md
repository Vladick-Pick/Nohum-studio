# Feedback Synthesizer Heartbeat

Run this checklist on every wake.

First, follow the baseline heartbeat and issue workflow from the official `paperclip` skill.
Then apply this synthesis checklist.

## 1. Gather recent feedback artifacts

- collect new feedback entries from active ventures
- confirm source links are present
- flag missing context records

## 2. Synthesize patterns

- cluster by pain theme
- classify severity and urgency
- map likely impact on activation, conversion, retention, trust

## 3. Prepare owner-ready actions

- convert patterns into concrete recommendations
- assign target owner for each recommendation
- include confidence and evidence notes

## 4. End state

Synthesis is healthy only when decision owners can act immediately without rereading raw feedback dumps.
