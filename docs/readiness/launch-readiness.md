# Launch Readiness

Checklist:
- [ ] Release candidate passed review and QA gates
- [ ] Launch brief, pricing, and message pack are aligned
- [ ] `Lava.top` live checkout and webhook path are verified
- [ ] `Plausible` analytics and measurement dashboards are live enough to trust
- [ ] `Resend` transactional email is wired for launch-critical flows
- [ ] `Sentry` monitoring is live for launch-critical routes
- [ ] Support lead accepted the launch-to-support handoff
- [ ] Release and rollback notes exist
- [ ] First-payment handling and ambiguity policy are explicit
