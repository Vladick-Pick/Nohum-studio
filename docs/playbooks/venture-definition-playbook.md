# Venture Definition Playbook

Owners:
- `Launch Lead`
- `Product Definer`
- `UX Researcher`
- `UX Architect`
- `UI Designer`
- `Pricing Strategist`

Expected outputs:
- definition packet
- launch-brief
- handoff dossier
- Gate B recommendation

Operating sequence:
1. Start from the research dossier and queue decision, not from a blank page.
2. Lock ICP, JTBD, offer, pricing, UX flow, and MVP boundary into canonical artifacts.
3. Surface unresolved assumptions explicitly and route them before Gate B.
4. Do not hand work to Engineering until the definition packet and acceptance criteria are coherent.
