# Launch Playbook

Owners:
- `Launch Lead`
- `CMO`
- `Growth Lead`
- `Launch Program Manager`

Expected outputs:
- launch plan
- campaign brief
- measurement dashboard
- launch status readout

Operating sequence:
1. Freeze the launch brief before channel execution begins.
2. Ensure pricing, messaging, `Lava.top` checkout, `Plausible` measurement, and support readiness match the actual product and release status.
3. Verify `Resend` transactional email and `Sentry` monitoring are live before traffic ramps.
4. Run channel execution through measurable experiments, not vanity activity.
5. Keep Support and Engineering informed of launch-day expectations and risks.
