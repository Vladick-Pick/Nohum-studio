# Org Hygiene Checklist

Check weekly:

- reporting lines match the intended org map
- no duplicate manager roles exist
- paused versus active state is intentional
- recurring manager tasks still exist and are assigned correctly
- budgets match current policy
- no critical transition is happening through comments-only context
- stale issues older than policy allow are rerouted or closed
- support, launch, and engineering lanes still point to canonical artifacts

Leave the result in `docs/templates/operations/org-hygiene-report.md` or an equivalent structured issue update.
